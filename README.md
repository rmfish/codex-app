# Codex App Readable Extract

This directory contains a curated subset of files extracted from Codex using Electron's official ASAR extractor.

Main / preload side:

- package.json
- .vite/build/\*.js

Renderer side:

- webview/assets/app-main-\*.js
- webview/assets/app-shell-\*.js
- webview/assets/app-session-\*.js
- webview/assets/app-server-\*.js
- webview/assets/artifact-tab-content.electron-\*.js
- selected settings, apps, and automation bundles
